# Веб-сайт наследия Нимы Пурбуева

Этот проект представляет собой современный веб-сайт, посвященный творчеству и биографии забайкальского художника Нимы Пурбуевича Пурбуева. Сайт выполнен в стиле «этнического романтизма» с использованием современных веб-технологий.

## Технологический стек

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (v3)
- **Animation**: Framer Motion
- **Icons**: Lucide React

## Структура проекта

```
src/
├── assets/         # Изображения и статические файлы
├── components/     # React компоненты
│   ├── home/       # Компоненты главной страницы (Hero, Gallery, Biography, etc.)
│   └── layout/     # Общие компоненты (Navbar, Footer, Layout)
├── data/           # (Зарезервировано) JSON данные
├── App.jsx         # Главный компонент приложения
└── index.css       # Глобальные стили и Tailwind директивы
```

## Установка и запуск

1.  **Клонирование репозитория:**
    ```bash
    git clone <repository-url>
    cd icy-galaxy
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    ```

3.  **Запуск в режиме разработки:**
    ```bash
    npm run dev
    ```
    Откройте браузер по адресу `http://localhost:5173`.

## Сборка для продакшена

Для создания оптимизированной сборки выполните:

```bash
npm run build
```

Файлы будут созданы в папке `dist`.

## Деплой на Netlify

1.  Зарегистрируйтесь на [Netlify](https://www.netlify.com/).
2.  Создайте новый сайт из Git (GitHub/GitLab/Bitbucket) или перетащите папку `dist` (после выполнения команды build) в область деплоя Netlify Drop.
3.  При настройке CI/CD укажите:
    - **Build command**: `npm run build`
    - **Publish directory**: `dist`

## Примечания

- **Изображения**: В текущей версии используются CSS-плейсхолдеры и градиенты, так как сервис генерации изображений был временно недоступен. Рекомендуется заменить их на реальные фотографии картин Нимы Пурбуева в высоком разрешении по мере их оцифровки.
- **Шрифты**: Используются Google Fonts (Playfair Display для заголовков, Inter для текста).
